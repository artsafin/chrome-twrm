Teamwox enhancements extension

How to use with Redmine:

  1. Configure Redmine API key in settings - that will enable most of the features

What's new:

Version 1.1

  1. Fixed bugs with slash
  2. Parent- & sub-tasking: preselect Project Redmine as parent inheriting crucual data form parent task
  3. Simplified Redmine layout - removed unnecessary headers and paddings

Version 1.0

  1. Create redmine to arbitrary project in Redmine - use Settings to configure project
  2. Creating redmine now doesn't show any intermediate windows - it directly shows native Redmine interface right in popup with prefilled fields
  3. After issue has been created this can be posted to Teamwox
  4. Creating redmines from Teamwox tasks is not supported anymore
  5. Creating project ("root") redmines:
    1. Servicedesk must reside in Teamwox project
    2. To create root redmine, select project name in head post of the servicedesk and press the button
  6. Improved subtasking! Created redmine tasks based on servicedesks under the same Teamwox project will be created as a subtask of the root redmine (root redmine subject must exactly match the name of the Teamwox project).
    Requires Redmine API Key configured in Settings
  7. Advanced magic possible using "Moar magic" option in settings
  8. The extension adds search to Servicedesk sections list:
    Use "/" for a keyboard shortcut to jump to search.
  9. (If API key is configured in settings) Redmine tooltip when mouse is over a Redmine link
